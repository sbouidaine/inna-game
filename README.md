# تحدي إن وأخواتها

لعبة ويب تعليمية تفاعلية لتدريب الطلاب على **إن وأخواتها** بطريقة سريعة وممتعة.

## فكرة اللعبة

- كل جولة فيها سؤالان:
  - اختيار الأداة الصحيحة (إن، أن، كأن، لكن، ليت، لعل)
  - تحديد إعراب الاسم أو الخبر
- يوجد عداد وقت عام (90 ثانية)
- يوجد 3 قلوب فقط
- السلسلة (Streak) ترفع النقاط مع كل إجابة صحيحة متتالية

## التشغيل محليًا

هذا مشروع ثابت (Static) بدون أي بناء (Build). يكفي فتح `index.html` في المتصفح.

## النشر على GitHub Pages

1. ارفع الملفات إلى المستودع (الفرع `main`):
   - `index.html`
   - `style.css`
   - `script.js`
2. في GitHub افتح:
   - Settings → Pages
3. في Source اختر:
   - Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
4. احفظ الإعدادات وانتظر دقيقة تقريبًا حتى يظهر رابط النشر.

## ملاحظات تصميم

- الواجهة تدعم العربية واتجاه RTL.
- التصميم متجاوب للهواتف والكمبيوتر.
- تمت إضافة حركات خفيفة مع احترام إعداد `prefers-reduced-motion`.
